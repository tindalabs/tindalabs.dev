# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✓         |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: ikerlaforga@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive an acknowledgement within **48 hours** and a resolution or status update within **90 days**.

Once a fix is released, the vulnerability will be disclosed publicly with credit to the reporter (unless you prefer to remain anonymous).
